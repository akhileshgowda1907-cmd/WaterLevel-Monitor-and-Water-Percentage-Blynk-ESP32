# Water Level Monitoring System using ESP32

## Project Description
This project monitors water level and water percentage using an ultrasonic sensor and ESP32.

## Components
- ESP32
- HC-SR04
- Blynk IoT

## Circuit Connections
- TRIG -> GPIO 5
- ECHO -> GPIO 18

## Arduino Code

```cpp
#define TRIG_PIN 5
#define ECHO_PIN 18
```

## Output
- Water level in cm
- Water percentage
